Represents a 2D (x,y) point.

## Properties

| Property | Type | Description |
|:---------|:-----|:------------|
<prop>x</prop> | <type>number</type> | X coordinate
<prop>y</prop> | <type>number</type> | Y coordinate

## Constructors

Creates a new point with <prop>x</prop> and <prop>y</prop> both set to `0`.
